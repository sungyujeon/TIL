# String method

(작성중)

- String(String s)

  ```java
  String s = new String("hello");  // hello
  ```

- String(char[] value)

  ```java
  char[] c = ['a', 'e']
  String s = new String(c);  // ae
  ```

- String(StringBuffer buf)

  ```java
  String buffer sb = new StringBuffer("hello");  //hello
  ```

- char charAt(int index)

  ```java
  "hello".charAt(1);  // e
  "aeiou".charAt(2);  // i
  ```
